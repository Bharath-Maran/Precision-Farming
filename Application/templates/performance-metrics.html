ACCURACY COMPARISON

1. Precision = TP / (TP + FP)
2. Recall = TP / (TP + FN)
3. F1 Score = 2 * (Precision * Recall) / (Precision + Recall)
4. Support: It is the number of occurrences of each class in the target variables
5. Accuracy = (TP + TN) / (TP + TN + FP + FN)
6. Accuracy of all models is plotted in bar chart
7. Highly accurate model is used for deployment
